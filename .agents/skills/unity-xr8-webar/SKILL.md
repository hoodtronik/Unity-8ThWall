# XR8WebAR Unity Addon — Skills Guide

## Overview

This addon provides 8th Wall AR tracking for Unity WebGL builds. It bridges C# scripts to the 8th Wall JS engine via `.jslib` files and `xr8-bridge.js`.

## Architecture

```
C# Scripts  ←→  .jslib (DllImport)  ←→  xr8-bridge.js  ←→  8th Wall Engine
```

## Quick Setup (One-Click)

1. **Menu:** `GameObject > XR8 WebAR > Complete AR Scene Setup`
2. This creates: AR Camera Rig, XR8Manager, XR8ImageTracker, and content placeholder
3. In `XR8Manager` inspector, check the tracking modes you need (Image, World, Face)
4. In `XR8ImageTracker` inspector, click `+ Add` next to discovered targets

## Adding Image Targets

1. Place target JSON + images in `Assets/image-targets/` (generated by 8th Wall Image Target Processor)
2. Select `XR8ImageTracker` in Hierarchy
3. Inspector auto-discovers targets — click `+ Add` button
4. Assign a Content Root transform (child GameObject where your AR content goes)
5. The image target will be visible in the Scene view as a textured quad

## Key Components

| Script | Purpose |
|---|---|
| `XR8Manager` | Unified controller — enable tracking modes here |
| `XR8Camera` | Camera feed background rendering |
| `XR8ImageTracker` | Image target tracking |
| `XR8FaceTracker` | Face tracking with expressions & landmarks |
| `XR8VideoController` | Video playback on targets |

## Testing

- **Phone:** Build WebGL → `npx serve Build/` → open URL on phone (same WiFi)
- **Desktop Preview:** Toggle in XR8Manager inspector (limited — events fire but video may not render)

## File Locations

- Runtime scripts: `Assets/XR8WebAR/Runtime/Scripts/`
- JS plugins: `Assets/XR8WebAR/Runtime/Plugins/`
- Editor scripts: `Assets/XR8WebAR/Editor/`
- JS bridge: `Assets/WebGLTemplates/8thWallTracker/xr8-bridge.js`
- HTML template: `Assets/WebGLTemplates/8thWallTracker/index.html`
- Image targets: `Assets/image-targets/`

## IMPORTANT: No App Keys

8th Wall engine is self-hosted. **There are NO app keys, NO cloud project IDs.** The engine binary lives at `Assets/WebGLTemplates/8thWallTracker/xr8.js`. Never ask for or try to configure an 8th Wall app key.
